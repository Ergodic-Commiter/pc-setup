|---------------|-----------|---------------|

| ctrl          | ^         |               |
| opt (win)     | ⌥ (⊞)     | meta in PC    |
| alt (cmd)     | ⎇(⌘)     | meta in Mac   |  
| fn            | fn        |               |
| tab           | ⇥         |               | 
| return        | ⏎         |               |
